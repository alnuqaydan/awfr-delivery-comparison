# AWFR - منصة مقارنة توصيل الطعام الشاملة

## نظرة عامة على المنصة

تم تطوير منصة **AWFR** (تطبيق توصيل الطعام الذكي) لتكون منصة شاملة لمقارنة خدمات توصيل الطعام، مما يساعد المستخدمين في العثور على أفضل العروض وأسرع أوقات التوصيل للوجبات المرغوبة.

## 🎯 الأهداف المحققة

### ✅ 1. تدفق المستخدم متعدد الخطوات
- **اكتشاف الموقع**: نظام ذكي لاكتشاف موقع المستخدم تلقائياً مع خيار التعديل اليدوي
- **اختيار المطعم**: تصفح المطاعم حسب الفئات والمواقع
- **تصفح القوائم**: عرض قوائم الطعام مع الفئات والقيود الغذائية
- **مقارنة التوصيل**: مقارنة شاملة لخيارات التوصيل المختلفة

### ✅ 2. فحص الأسعار والتوفر في الوقت الفعلي
- نظام ذكي لحساب أسعار التوصيل بناءً على:
  - المسافة الجغرافية
  - عوامل الازدحام المروري الحالية
  - مستوى الطلب (حسب الوقت واليوم)
  - رسوم الخدمة المتغيرة
- كاش ذكي للبيانات (5 دقائق)
- تحديث الأسعار كل 30 ثانية

### ✅ 3. شفافية الأسعار الكاملة
- تفصيل شامل لجميع الرسوم:
  - سعر الوجبة الأساسي
  - رسوم التوصيل
  - رسوم الخدمة
  - رسوم الطلب الصغير
  - رسوم المعالجة
  - الضريبة (15%)
- **التكلفة الإجمالية** واضحة ومقدمة مقدماً
- لا توجد رسوم خفية

### ✅ 4. مقارنة أوقات التوصيل المقدرة
- حساب دقيق بناءً على:
  - الظروف المرورية الحالية
  - المسافة من المطعم
  - أوقات التحضير المتوقعة
  - تاريخ أداء الشركة

## 🏗️ هيكل الروابط المحسن

### صفحة الاستكشاف
```
/explore?lat={latitude}&lng={longitude}&address={encoded_address}&radius={distance}&cuisine={type}&is_open={boolean}
```

### صفحة المطعم
```
/restaurant/{restaurant-slug}?restaurant_id={id}&lat={lat}&lng={lng}&menu_category={category}&dietary={restrictions}
```

### صفحة مقارنة التوصيل
```
/delivery-options?restaurant_id={id}&item_id={id}&item_name={encoded_name}&base_price={price}&user_lat={lat}&user_lng={lng}&compare_all=true
```

## 🎨 تجربة المستخدم المحسنة

### للأجهزة المحمولة
- تصميم **Mobile-First** محسن للقرارات السريعة
- مكونات مضغوطة مع المعلومات الأساسية
- أزرار CTA بارزة وسهلة الضغط
- تفاعل باللمس محسن (swipe, tap)

### أداة اتخاذ القرار السريع
- **عداد تنازلي 60 ثانية** لتشجيع القرار السريع
- **توصيات ذكية** بناءً على:
  - وقت اليوم
  - توازن السعر/الوقت/الموثوقية
  - تفضيلات المستخدم المحتملة
- **إحصائيات سريعة** للمقارنة الفورية
- أزرار **الأرخص** و **الأسرع** للاختيار السريع

### ميزات التحليل المتقدم
- **نظام النقاط الذكي** لتقييم الخيارات
- **حساب التوفير** (مال ووقت)
- **أوسمة ديناميكية** (أسرع، أرخص، أعلى تقييماً، موصى به)
- **عوامل الوقت الفعلي** (ازدحام، طلب)
