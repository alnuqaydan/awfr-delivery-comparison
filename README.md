# AWFR - منصة توصيل الطعام الذكية

## نظرة عامة

AWFR هي منصة توصيل طعام شاملة ومتطورة تتيح للمستخدمين مقارنة أسعار خدمات التوصيل المختلفة واختيار أفضل خيار يناسبهم من حيث السعر والسرعة. تم تطوير المنصة باستخدام أحدث تقنيات الويب مع التركيز على تجربة المستخدم المثلى.

## الميزات الرئيسية

### 🏪 صفحة المطعم
- عرض تفاصيل شاملة للمطعم (الاسم، الصورة، التقييم، أوقات العمل)
- قائمة منتجات مقسمة حسب الفئات مع الصور والأسعار
- إضافة المنتجات إلى السلة مع تخصيص الكمية
- واجهة مستخدم متجاوبة ومتطورة

### 🛒 إدارة السلة
- عرض واضح للمنتجات المختارة
- تعديل الكميات وحذف المنتجات
- حساب المجموع الفرعي والضرائب والمجموع الإجمالي
- الانتقال السلس إلى مرحلة مقارنة التوصيل

### 🚚 مقارنة خدمات التوصيل
- ربط سلة التسوق بنظام مقارنة خدمات التوصيل
- حساب تكاليف التوصيل المختلفة بناءً على المطعم والموقع
- عرض خيارات التوصيل المتاحة مع الأسعار والأوقات
- تحديد الأرخص والأسرع تلقائياً

### 📍 اكتشاف الموقع التلقائي
- استخدام Geolocation API لاكتشاف موقع المستخدم
- حساب المسافات الفعلية بين المطعم والعميل
- تحسين دقة أسعار التوصيل

### 🌐 دعم متعدد اللغات
- دعم اللغة العربية والإنجليزية
- واجهة مستخدم محلية بالكامل
- تخطيط RTL للغة العربية

## التقنيات المستخدمة

### Frontend
- **Next.js 15** - إطار عمل React متطور
- **React 18** - مكتبة واجهة المستخدم
- **TypeScript** - لكتابة كود آمن ومنظم
- **Material-UI (MUI)** - مكتبة مكونات UI
- **Tailwind CSS** - إطار عمل CSS
- **Redux Toolkit** - إدارة حالة التطبيق

### Backend & State Management
- **Redux Toolkit** - إدارة حالة التطبيق
- **React Redux** - ربط React مع Redux
- **RTK Query** - إدارة الطلبات والبيانات

### التطوير والاختبار
- **Vitest** - إطار عمل الاختبار
- **Playwright** - اختبارات E2E
- **ESLint** - فحص جودة الكود
- **Prettier** - تنسيق الكود

```
│   └── ...
├── store/                 # إدارة الحالة
│   ├── index.ts           # إعداد المتجر
│   ├── restaurantSlice.ts # حالة المطاعم
│   ├── cartSlice.ts       # حالة السلة
│   ├── pricingSlice.ts    # حالة الأسعار
│   └── ...
├── data/                  # البيانات الثابتة
│   └── restaurants.ts     # بيانات المطاعم
├── utils/                 # الأدوات المساعدة
│   └── pricing.ts         # حساب الأسعار
└── types/                 # تعريفات TypeScript
    └── index.ts           # أنواع البيانات
```

## التثبيت والتشغيل

### المتطلبات
- Node.js 18 أو أحدث
- npm أو yarn

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone <repository-url>
cd awfr
```

2. **تثبيت التبعيات**
```bash
npm install
```

3. **تشغيل المشروع في وضع التطوير**
```bash
npm run dev
```

4. **فتح المتصفح**
```
http://localhost:3000
```

### أوامر مفيدة

```bash
# تشغيل في وضع التطوير
npm run dev

# بناء المشروع للإنتاج
npm run build

# تشغيل المشروع المبنى
npm run start

# تشغيل الاختبارات
npm run test

# تشغيل اختبارات E2E
npm run test:e2e

# فحص جودة الكود
npm run lint

# فحص الأنواع
npm run type-check
```

### إدارة الحالة
