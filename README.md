# AWFR - تطبيق توصيل الطعام

تطبيق شامل لتوصيل الطعام يربط بين المطاعم والعملاء مع مقارنة أسعار شركات التوصيل المختلفة.

## الميزات الرئيسية

### 🍽️ إدارة المطاعم
- عرض قائمة شاملة بالمطاعم المحلية
- تصنيف المطاعم حسب نوع المأكولات (عربية، آسيوية، إيطالية، حلويات، مشروبات)
- تقييمات ومراجعات العملاء
- معلومات التوصيل والحد الأدنى للطلب

### 🛒 نظام الطلبات
- سلة تسوق تفاعلية
- إدارة الكميات والإضافات
- تتبع حالة الطلب
- تاريخ الطلبات السابقة

### 🚚 مقارنة شركات التوصيل
- مقارنة أسعار شركات التوصيل المختلفة
- حساب تكلفة التوصيل حسب المسافة
- اختيار أفضل شركة توصيل
- روابط مباشرة لشركات التوصيل

### 🌐 دعم متعدد اللغات
- واجهة باللغتين العربية والإنجليزية
- ترجمة شاملة لجميع المحتويات
- دعم اتجاه النص (RTL/LTR)

### 📱 تصميم متجاوب
- واجهة محسنة للأجهزة المحمولة
- تصميم حديث وسهل الاستخدام
- تجربة مستخدم سلسة



## التقنيات المستخدمة

### Frontend
- **Next.js 14** - إطار عمل React
- **TypeScript** - لغة البرمجة
- **Material-UI (MUI)** - مكتبة واجهة المستخدم
- **Tailwind CSS** - إطار عمل CSS
- **React i18next** - الترجمة الدولية

### State Management
- **Redux Toolkit** - إدارة الحالة
- **React Hooks** - إدارة المكونات

### Testing
- **Jest** - اختبار الوحدات
- **Playwright** - اختبار النهاية للنهاية
- **Vitest** - إطار اختبار سريع

### Development Tools
- **ESLint** - فحص الكود
- **Prettier** - تنسيق الكود
- **Husky** - Git hooks

## التثبيت والتشغيل

### المتطلبات
- Node.js 18+ 
- npm أو yarn

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone <repository-url>
cd Awfer
```

2. **تثبيت التبعيات**
```bash
npm install
# أو
yarn install
```

3. **تشغيل التطبيق في وضع التطوير**
```bash
npm run dev
# أو
yarn dev
```

4. **فتح المتصفح**
```
http://localhost:3000
```

### أوامر مفيدة

```bash
# بناء التطبيق للإنتاج
npm run build

# تشغيل التطبيق المبنى
npm run start

# تشغيل الاختبارات
npm run test

# تشغيل اختبارات النهاية للنهاية
npm run test:e2e

# فحص الكود
npm run lint

# تنسيق الكود
npm run format
```
